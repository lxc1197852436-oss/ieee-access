# 第4章 语言增强深度强化学习方法

## 4.1 本章概述

第3章已经构建了面向中国虚拟电厂场景的数据体系和仿真环境。本章在此基础上设计语言增强深度强化学习方法。核心思想是：使用文本语义编码器将电力市场公告、气象预警、需求响应通知和新能源消纳提示转化为连续语义特征，再与负荷、光伏、电价、温度和储能 SOC 等数值状态融合，输入最大熵强化学习算法 Soft Actor-Critic（SAC），从而学习连续储能充放电策略[21-24, C1]。

与中期阶段的 `LE-DRL-Semantic` 启发式策略不同，本章提出的正式方法是可训练的 LE-DRL-SAC。其训练目标不是简单根据阈值充放电，而是在虚拟电厂环境中通过交互最大化长期累计收益，并在策略熵正则项约束下保持探索能力[9, 21, 22, C1]。

## 4.2 方法总体框架

本文方法由四个模块组成：

1. 数值状态编码模块：对负荷、光伏、电价、温度、SOC 和时间特征进行归一化。
2. 文本语义编码模块：将中文文本事件映射为语义风险向量。
3. 状态融合模块：拼接数值状态与语义状态，构造增强状态。
4. SAC 决策模块：基于增强状态输出连续储能充放电功率。

![图4-1 LE-DRL-SAC模型架构](../outputs/thesis_figures/fig3_ledrl_sac_model_architecture.png)

图4-1给出了 LE-DRL-SAC 模型结构。数值输入经过归一化形成 `s_num`，中文文本事件经过 DeepSeek 或本地语义编码器形成 `s_sem`，二者拼接为增强状态 `s_aug` 后输入 SAC Actor-Critic 网络。需要强调的是，大语言模型不直接输出储能动作，其输出只作为语义状态和解释依据，最终动作由可复现的 SAC Actor 产生。

标准 SAC 基线仅使用数值状态：

```text
s_num,t = [load_t, pv_t, price_t, temperature_t, SOC_t, sin(hour_t), cos(hour_t)]
```

LE-DRL 使用增强状态：

```text
s_aug,t = [s_num,t, s_sem,t]
```

其中 `s_sem,t` 包括风险分数、价格尖峰分数、负荷压力分数、新能源消纳压力分数和风险平方根特征。后续如果接入主流大模型或中文预训练模型，该语义向量可替换为更高维的 embedding。

## 4.3 数值状态编码

为了避免不同物理量量纲差异影响神经网络训练，本文对数值状态进行归一化。电价以 300 元/MWh 为中心、220 元/MWh 为尺度进行标准化；负荷按 6 MW 缩放；光伏按 5 MW 缩放；温度以 30 摄氏度为中心、8 摄氏度为尺度；SOC 保持 0 到 1 的自然范围。时间特征采用正弦和余弦编码，以表达日周期连续性：

```text
time_sin = sin(2π hour / 24)
time_cos = cos(2π hour / 24)
```

数值状态编码对应代码：

`app/core/rl/state_encoder.py`

## 4.4 文本语义编码

本文将文本事件分为高温预警、需求响应、价格尖峰和新能源消纳四类。该分类对应虚拟电厂运行中常见的天气负荷风险、市场价格风险、需求响应和新能源消纳压力[1-3, 13, 38]。当前系统使用本地规则语义编码器，输出以下五维特征：

```text
s_sem,t = [risk, price_spike, load_pressure, renewable_curtailment, sqrt(risk)]
```

其中，`risk` 表示综合风险水平，`price_spike` 表示价格尖峰风险，`load_pressure` 表示负荷压力，`renewable_curtailment` 表示新能源消纳压力。该设计保证在没有 API key 或大模型环境时系统仍可复现。后续接入 API 后，可将 `event_text` 输入主流大模型或中文领域模型，获得更细粒度的风险预测和自然语言解释。

需要强调的是，本文不建议让大模型直接输出最终调度动作。调度动作仍由可复现的强化学习策略产生，大模型主要用于语义理解和可解释分析[23, 24, 26, 28]。

## 4.5 Soft Actor-Critic 算法

SAC 是一种适用于连续动作空间的最大熵 Actor-Critic 算法。与传统强化学习只最大化累计奖励不同，SAC 同时最大化奖励和策略熵[21, 22, C1]：

```text
J(π) = Σ E[r(s_t, a_t) + α H(π(·|s_t))]
```

其中，`α` 为熵温度系数，`H` 为策略熵。熵正则项能够鼓励智能体探索更多调度动作，在电价波动、新能源出力不确定和文本事件突发的虚拟电厂场景中具有较好适用性。

SAC 使用高斯策略网络输出动作分布：

```text
a_t = tanh(μ_θ(s_t) + σ_θ(s_t) ε) * a_max
```

其中 `ε` 为标准正态噪声，`tanh` 用于将动作限制在 `[-a_max, a_max]`。本文中 `a_max = 2 MW`。

Critic 部分采用双 Q 网络以缓解过估计问题：

```text
Q_min(s, a) = min(Q_1(s, a), Q_2(s, a))
```

目标 Q 值为：

```text
y = r + γ(1-d)[min(Q'_1(s', a'), Q'_2(s', a')) - α log π(a'|s')]
```

Actor 的优化目标为：

```text
J_π = E[α log π(a|s) - min(Q_1(s,a), Q_2(s,a))]
```

## 4.6 LE-DRL-SAC 算法设计

LE-DRL-SAC 与标准 SAC 使用相同的 Actor-Critic 训练过程，区别在于输入状态。标准 SAC 输入 `s_num,t`，LE-DRL-SAC 输入 `s_aug,t`。该设计保证两者在网络结构、动作空间、奖励函数和训练超参数上保持一致，使实验对比更公平。

本文不将大语言模型作为最终调度器。AI 模块在方法中承担两个角色：第一，将市场公告、气象预警和调度通知等中文文本转化为风险等级、价格尖峰压力、负荷压力和新能源消纳压力等结构化语义特征；第二，在策略执行后结合状态、动作和语义特征生成可读的决策解释。储能充放电动作仍由可复现的强化学习策略输出，避免大语言模型直接给出调度动作所带来的随机性和不可复现实验问题[23-28]。

为使语义特征不仅停留在状态拼接层，本文进一步在 LE-DRL-SAC 的 Actor 训练中加入语义一致性正则项。该正则项根据当前可观测状态和文本风险特征构造风险一致的动作参考方向：当价格尖峰或负荷压力较高时，鼓励 Actor 学习高价放电或保留高价套利能力；当新能源消纳压力或低价充电机会较强时，鼓励 Actor 学习充电吸纳新能源。该项只参与训练损失，不在测试阶段直接覆盖 Actor 动作，因此最终评估动作仍由训练后的 SAC Actor 产生。

加入语义一致性项后，Actor 目标可写为：

```text
J_π^LE = E[α log π(a|s_aug) - min(Q_1(s_aug,a), Q_2(s_aug,a))]
         + λ_sem ||a - a_sem(s_aug)||²
```

其中，`a_sem(s_aug)` 为由价格、SOC 和文本风险分数构造的语义一致性参考动作，`λ_sem` 为语义一致性权重。本文正式实验中使用 `λ_sem = 5.0`，并在测试阶段设置 `semantic_guidance_weight = 0`，保证测试动作不由规则或大语言模型直接修正。

LE-DRL-SAC 流程如下：

1. 读取当前 VPP 数值状态和中文文本事件。
2. 对数值状态进行归一化。
3. 对文本事件进行语义编码。
4. 拼接得到增强状态。
5. Actor 输出连续充放电动作。
6. 环境执行动作并返回奖励与下一状态。
7. 将经验写入 replay buffer。
8. 使用 SAC 目标函数、语义辅助奖励和 Actor 语义一致性正则更新 Actor 与 Critic。

## 4.8 与中期基线的关系

中期阶段已经实现了 `Soft-Q-Numeric` 和 `Soft-Q-Semantic`，用于证明强化学习闭环能够运行。但 Soft-Q 采用离散动作和状态分箱，无法充分体现虚拟电厂储能连续功率控制特性[4, C3]。本章的 SAC 与 LE-DRL-SAC 是正式论文阶段的核心算法，将替代 Soft-Q 成为第6章实验对比的主要模型[21, 22, C1]。

中期策略与正式模型关系如下：

| 阶段 | 模型 | 用途 |
|---|---|---|
| 中期 | Rule-Based | 基础规则基线 |
| 中期 | LE-DRL-Semantic | 语义启发式展示策略 |
| 中期 | Soft-Q | 轻量可训练RL闭环 |
| 正式 | SAC-Numeric | 仅数值状态强化学习基线 |
| 正式 | LE-DRL-SAC | 本文核心方法 |
| 正式 | LE-DRL w/o Text | 文本消融实验 |

## 4.9 本章小结

本章提出了面向虚拟电厂动态优化的语言增强深度强化学习方法。该方法通过文本语义编码器将中文电力事件转化为语义风险向量，并与数值状态融合形成增强状态；在决策层采用最大熵 SAC 学习连续储能充放电策略。相较传统仅使用数值状态的 SAC，LE-DRL-SAC 能够利用市场公告、气象预警、需求响应和新能源消纳提示等文本信息，为后续极端场景下的前瞻性调度和可解释分析提供基础。
