# 第1章 绪论

## 1.1 研究背景与意义

随着新型电力系统建设持续推进，电力系统运行形态正在由传统的“源随荷动”逐步转向源网荷储多主体协同互动。高比例新能源接入提升了能源结构清洁化水平，也带来了出力波动性、预测不确定性和系统调节压力。与此同时，电力现货市场、需求响应、分布式资源聚合和用户侧储能等机制不断发展，使得配电侧和用户侧资源从被动用能对象逐步转变为可调度、可交易、可优化的灵活性资源。

虚拟电厂（Virtual Power Plant, VPP）正是在这一背景下形成的重要技术形态。虚拟电厂通过信息通信、聚合控制和优化调度技术，将分布式光伏、储能、可调负荷、电动汽车和其他分布式能源资源聚合为统一的可调控单元，并参与电力市场、辅助服务和需求响应[1-3]。与单一储能或单一负荷控制不同，虚拟电厂运行需要同时面对电价变化、负荷波动、新能源出力不确定、储能寿命约束以及市场规则变化等多重因素，因此其动态优化与决策问题具有明显的时序性、约束性和不确定性[8, 31, 44]。

从国内公开数据看，中国电力需求和新能源规模均保持较高水平。根据国家能源局公开数据，2024 年全社会用电量为 98521 亿千瓦时，同比增长 6.8%；同年全国可再生能源累计装机达到 18.89 亿千瓦，其中风电装机 5.21 亿千瓦，太阳能发电装机 8.87 亿千瓦，风电和太阳能发电量合计达到 1.83 万亿千瓦时[15]。这些数据表明，电力系统一方面面临持续增长的用电需求，另一方面也面临高比例新能源并网带来的调节需求。对于包含分布式光伏和储能的虚拟电厂而言，如何在电力市场价格波动和新能源消纳压力下实现经济运行和风险控制，已经成为具有现实意义的研究问题[16-18]。

在现货市场环境下，虚拟电厂的调度决策不仅依赖负荷、光伏、电价、温度和储能荷电状态（State of Charge, SOC）等数值信息，还受到气象预警、市场公告、需求响应通知、新能源消纳提示等文本信息影响。例如，高温预警往往意味着晚高峰空调负荷上升，价格尖峰公告可能提示市场套利机会，新能源消纳压力提示午间储能充电需求增强。这类信息通常以自然语言文本形式出现，传统优化模型和常规强化学习模型难以直接利用。如果忽略文本事件，模型可能只根据当前数值状态做出反应，缺少对未来风险和调度意图的提前感知。

近年来，大语言模型（Large Language Model, LLM）在自然语言理解、信息抽取和结构化推理方面表现出较强能力，为电力系统文本信息利用提供了新的技术路径[23-28]。与此同时，深度强化学习（Deep Reinforcement Learning, DRL）能够在复杂动态环境中通过交互学习序贯决策策略，适合处理虚拟电厂储能充放电、需求响应和市场交易等连续决策问题[9, 21, 22, 29]。因此，将大语言模型的文本语义理解能力与深度强化学习的动态决策能力结合，有望提升虚拟电厂在复杂事件场景下的调度适应性和可解释性。

基于上述背景，本文围绕“大语言模型与深度强化学习融合的虚拟电厂动态优化与决策”开展研究。研究重点不是让大语言模型直接输出最终调度动作，而是将大语言模型或语义编码器用于文本事件理解，将气象预警、市场公告和调度提示转化为可供强化学习利用的语义状态特征；最终调度动作仍由可复现、可训练、可评价的深度强化学习策略输出。这样的设计兼顾了文本理解能力、调度可复现性和实验可评价性。

## 1.2 国内外研究现状概述

虚拟电厂优化调度研究长期受到国内外学者关注。现有研究通常从资源聚合、市场交易、需求响应、储能优化和不确定性建模等角度展开[1-3, 8, 50]。传统方法多采用数学规划、鲁棒优化、随机优化或模型预测控制等方式，在已知或可预测的负荷、电价和新能源出力条件下求解调度计划[10-12, 40]。这类方法具有较强的可解释性和约束处理能力，但在面对高维状态、复杂非线性关系和频繁变化的市场环境时，模型构建和求解成本较高。

强化学习为虚拟电厂动态调度提供了另一类思路。强化学习不要求预先给出完整的系统转移模型，而是通过智能体与环境交互学习状态到动作的映射[9, 29, C3]。对于储能充放电和需求响应等序贯决策问题，强化学习能够根据历史经验逐步学习不同电价、负荷和新能源场景下的调度策略。尤其是 Soft Actor-Critic（SAC）等适用于连续动作空间的深度强化学习算法，能够直接输出连续储能功率，较适合虚拟电厂储能控制问题[21, 22, C1]。

不过，现有强化学习调度研究多数仍以数值状态为主，状态空间通常包含负荷、电价、新能源出力、SOC、时间等变量，对自然语言事件的利用不足。在真实电力系统运行中，调度公告、气象预警、市场通知和政策信息往往以文本形式出现，且会影响负荷趋势、价格预期和调度目标。如果模型无法理解这些文本信息，可能导致其在极端天气、价格尖峰和新能源消纳压力场景下缺少前瞻性响应。

大语言模型的发展为该问题提供了新的解决思路。大语言模型能够对中文文本进行语义理解、风险识别和结构化信息抽取，可将非结构化事件转化为风险分数、价格趋势、负荷压力和新能源消纳压力等特征。将这些语义特征与数值状态融合后输入强化学习模型，可以构建语言增强深度强化学习框架。该框架既保留强化学习在动态决策方面的优势，又引入大语言模型对文本事件的理解能力。

总体来看，现有研究在虚拟电厂优化调度、深度强化学习控制和大语言模型文本理解方面均已形成一定基础，但将中文电力事件文本、大语言模型语义理解和连续动作强化学习统一用于虚拟电厂动态优化的研究仍有进一步探索空间。特别是在国内公开数据有限的条件下，如何构建可复现的数据管线、合理说明数据边界，并通过仿真环境验证语言增强状态对调度决策的影响，是本文需要解决的关键问题。

## 1.3 研究内容

本文以包含分布式光伏、储能和聚合负荷的虚拟电厂为研究对象，围绕中文文本事件增强的动态优化决策方法开展研究。主要研究内容包括以下几个方面。

第一，构建面向中国虚拟电厂场景的数据体系。考虑到国内真实虚拟电厂 15 分钟级运行数据和完整电力现货出清数据通常不公开，本文采用“国内公开披露数据校准 + 虚拟电厂仿真构造”的方式建立实验数据。公开数据用于说明电力需求、新能源发展和广东电力市场背景；仿真数据用于构造强化学习交互环境。当前已整理国家能源局 2024 年用电量、可再生能源装机与发电量数据，以及广东电力现货和独立储能公开披露信息，并据此生成广东虚拟电厂 15 分钟样例场景。

第二，建立虚拟电厂动态优化环境。本文将虚拟电厂调度建模为马尔可夫决策过程，状态包括负荷、光伏出力、电价、温度、SOC、时间特征和文本事件语义特征；动作为空间储能充放电功率；奖励函数综合考虑购售电收益、储能退化成本、弃光惩罚和动作不可执行惩罚。该环境能够支持规则策略、强化学习策略和语言增强强化学习策略在统一条件下进行比较。

第三，设计语言增强深度强化学习方法。本文采用 Soft Actor-Critic 作为连续动作强化学习基础算法，构建数值状态基线 SAC-Numeric 和语言增强模型 LE-DRL-SAC。文本语义部分将高温预警、需求响应、价格尖峰和新能源消纳等中文事件编码为风险分数、价格尖峰分数、负荷压力分数和新能源消纳压力分数，并与数值状态拼接形成增强状态。为检验文本语义贡献，本文进一步设计 LE-DRL w/o Text 消融模型。

第四，实现可复现实验系统与展示平台。本文基于 Python 实现数据处理、虚拟电厂环境、强化学习训练、指标计算和结果导出，并基于 FastAPI 和本地 Dashboard 构建展示系统。系统支持场景生成、策略运行、收益指标展示、SOC 与储能动作轨迹可视化，以及文本事件和 AI 解释结果展示。

第五，开展多场景实验与结果分析。本文设计常规夏季运行、高温负荷压力、价格尖峰和新能源消纳压力四类场景，对 Rule-Based、SAC-Numeric、LE-DRL-SAC、LE-DRL w/o Text 等模型进行比较。评价指标包括总收益、CVaR 5%、储能吞吐量、高价放电率、低价充电率、终止 SOC 和轨迹行为等。当前阶段已完成 50 episode、3 随机种子的长训练实验，并生成文本消融结果和 SOC/动作轨迹图。

## 1.4 技术路线

本文技术路线可概括为“公开数据校准—虚拟电厂环境构建—文本语义编码—强化学习训练—多场景评估—系统展示”六个步骤。

![图1-1 论文总体技术架构](../outputs/thesis_figures/fig1_overall_research_architecture.png)

图1-1展示了本文总体技术架构。系统首先基于国内公开数据构建虚拟电厂仿真环境，再利用 DeepSeek 或本地语义编码器将中文文本事件转化为结构化风险特征，随后将数值状态与语义状态共同输入 LE-DRL-SAC 模型进行训练。最终通过多场景、多随机种子和文本消融实验评价模型收益、风险和储能行为，并输出论文图表和展示系统结果。

首先，收集和整理国内公开数据。通过国家能源局、广东电力市场和相关公开披露信息获取用电量、新能源装机、发电量、现货价格范围和储能交易信息，形成结构化数据表，并保存原始网页和文本记录，保证数据来源可追溯。

其次，构建虚拟电厂 15 分钟仿真场景。基于公开数据校准价格范围和运行背景，生成包含负荷、光伏、电价、温度和文本事件的连续时间序列。文本事件包括高温预警、需求响应、价格尖峰和新能源消纳等类型，用于模拟虚拟电厂实际运行中可能接收的外部信息。

再次，建立 VPP 强化学习环境。环境在每个调度步读取当前状态，执行储能充放电动作，更新 SOC，并根据收益、退化成本、弃光惩罚和动作约束计算奖励。该环境为后续规则策略、SAC 和 LE-DRL-SAC 提供统一实验平台。

然后，构建文本语义编码与状态融合模块。无 API key 时，系统使用本地规则语义编码器输出可复现的语义分数；接入主流 AI API 后，可通过 OpenAI-compatible 接口将中文事件转化为结构化语义特征。语义特征与数值状态拼接后形成增强状态，输入 LE-DRL-SAC。

最后，训练并评估调度策略。本文对数值状态 SAC、语言增强 LE-DRL-SAC 和文本消融模型进行训练，并在四类场景下比较其经济性、风险性和储能行为。实验结果通过表格和图形输出，并进一步接入 Dashboard 用于答辩展示。

## 1.5 主要创新点

本文当前形成的创新点主要体现在以下方面。

第一，构建了面向中国虚拟电厂场景的公开数据校准实验管线。针对国内真实虚拟电厂高时间分辨率数据难以公开获取的问题，本文明确区分公开披露数据和仿真构造数据，用公开数据校准研究背景、价格范围和运行参数，在此基础上构建可复现的虚拟电厂强化学习环境。

第二，将中文文本事件纳入虚拟电厂强化学习状态空间。本文将高温预警、需求响应、价格尖峰和新能源消纳等文本事件转化为语义风险特征，使模型能够在数值状态之外利用外部事件信息。这一设计有助于弥补传统数值状态调度模型对自然语言信息利用不足的问题。

第三，设计了语言增强 SAC 调度框架。本文将文本语义理解模块与 Soft Actor-Critic 连续动作强化学习结合，构建 LE-DRL-SAC 模型，使储能充放电动作仍由可训练、可复现的强化学习策略输出，而不是由大语言模型直接生成不可控动作。

第四，构建了包含训练、评估、消融和展示的完整实验系统。系统支持多场景、多模型、多随机种子实验，能够输出总收益、CVaR、储能行为、SOC 轨迹和文本消融结果，为中期答辩和后续学位论文实验分析提供支撑。

## 1.6 论文结构安排

本文共分为七章，各章安排如下。

第1章为绪论，介绍研究背景与意义、研究现状概述、研究内容、技术路线、主要创新点和论文结构。

第2章为相关理论与研究现状，重点介绍虚拟电厂优化调度、深度强化学习、Soft Actor-Critic、大语言模型语义理解以及语言增强决策相关研究。

第3章为数据体系与虚拟电厂环境建模，说明国内公开数据来源、数据处理流程、虚拟电厂系统结构、状态空间、动作空间、奖励函数和环境校验结果。

第4章为语言增强深度强化学习方法，介绍数值状态编码、文本语义编码、状态融合机制、SAC 算法原理和 LE-DRL-SAC 算法流程。

第5章为系统实现与实验设计，介绍实验平台架构、后端服务、前端展示、AI 接口设计、实验场景、对比模型和评价指标。

第6章为实验结果与分析，展示多场景实验结果、长训练结果、文本消融实验、SOC 与储能动作轨迹，并分析语言增强状态对虚拟电厂调度决策的影响。

第7章为总结与展望，总结全文研究工作和阶段性结论，分析当前数据和模型局限，并展望真实虚拟电厂数据、领域大模型、多智能体协同和更强优化基准等后续研究方向。

## 1.7 本章小结

本章围绕大语言模型与深度强化学习融合的虚拟电厂动态优化问题，说明了研究背景、现实意义、研究内容和技术路线。面对高比例新能源接入、电力市场价格波动和外部文本事件影响，传统仅依赖数值状态的调度模型难以充分利用气象预警、市场公告和需求响应通知等自然语言信息。本文拟通过构建公开数据校准的虚拟电厂仿真环境，将中文文本事件编码为语义状态，并与 SAC 连续动作强化学习结合，形成语言增强深度强化学习调度框架。后续章节将围绕数据体系、环境建模、算法设计、系统实现和实验结果展开详细论述。
