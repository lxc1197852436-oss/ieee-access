# 优先级1：国内公开数据源评估

## 结论先行

当前最适合本论文的国内数据路线是：

**广东电力现货/储能市场公开披露 + 国家能源局宏观电力数据 + 国家能源局可再生能源数据 + 中文市场/气象/调度文本事件 + 仿真 VPP 内部源荷储数据。**

原因：

- 你的题目核心是“LLM + DRL 融合的虚拟电厂动态优化与决策”，需要 15 分钟或小时级状态序列。
- 国内公开网页很难直接获得完整的真实 VPP 15 分钟运行数据。
- 广东已有电力现货市场和独立储能公开披露，最适合校准电价范围、储能价差和市场化叙事。
- 国家能源局数据权威性高，适合写背景、研究意义和参数合理性，不适合直接训练 RL。

## 数据源适配性排序

### 1. 广东现货市场与储能公开披露

最适合你的论文。

可用于：

- 价格范围校准：现货日前价格波动区间。
- 储能商业模式：独立储能充放电价差和结算价格。
- 论文实验场景：广东省 VPP 市场化运行。
- 文本事件：市场公告、现货波动、需求响应、储能参与现货市场。

不足：

- 公开页面是汇总披露，不是完整 15 分钟出清序列。
- 需要配合仿真生成连续时间序列。

### 2. 国家能源局 2024 年全社会用电量数据

适合写论文背景和负荷侧宏观校准。

可用于：

- 证明中国电力需求增长。
- 支撑“双碳目标与新型电力系统”的现实背景。
- 作为负荷仿真增长率或场景规模依据。

不足：

- 年度/分产业汇总，不能直接训练强化学习模型。

### 3. 国家能源局 2024 年可再生能源并网运行数据

适合写新能源占比、光伏风电增长和消纳压力背景。

可用于：

- 光伏/风电容量参数的合理性说明。
- 新能源消纳文本事件设计。
- 论文绪论和实验设定依据。

不足：

- 汇总数据，不含省级 15 分钟光伏/风电出力。

### 4. 新能源消纳监测数据

理论上很适合，但当前脚本访问的第三方转载页返回 403。为避免项目卡住，当前采用国家能源局官方全国风电/光伏利用率作为替代背景数据，并在环境中用剩余光伏功率动态刻画逐时消纳压力。

可用于：

- 各省风电/光伏利用率。
- 构造“新能源消纳压力”事件。
- 选择广东、山东、甘肃、新疆等不同消纳压力场景。

当前替代方案：

- 使用 `data/processed/new_energy_utilization_substitute_2024.csv`。
- 全国风电平均利用率：95.9%。
- 全国光伏发电利用率：96.8%。
- 省级逐时消纳压力由环境内部计算，不表述为真实省级消纳率时序数据。

### 5. 气象数据与预警文本

气象数据对你的论文非常重要，但国内公开气象历史数据通常需要注册或 key。

建议策略：

- 第一版用 Open-Meteo 的中国坐标历史天气作为无 key 备用数据。
- 正式论文如果需要“国内数据源”表述，可使用中国气象数据网、和风天气、心知天气等，需要账号或 key。
- 气象预警文本可作为 LLM 语义输入，不一定必须每条都是真实历史事件，论文中要标注“按真实预警格式构造”。

## 推荐论文数据组合

| 模态 | 推荐来源 | 是否直接入模型 | 说明 |
|---|---|---:|---|
| 电价 | 广东现货市场公开披露 + 仿真校准 | 是 | 当前最适合 |
| 负荷 | 国家能源局背景 + 广东典型负荷仿真 | 是 | 需要构造 15 分钟序列 |
| 光伏 | 可再生能源装机背景 + 辐照模型/仿真 | 是 | 可用于源荷储场景 |
| 温度 | Open-Meteo 中国坐标或国内天气 API | 是 | 高温预警和负荷尖峰相关 |
| 文本事件 | 市场公告、气象预警、调度通知 | 是 | 你的 LLM/语义增强核心 |
| 储能参数 | 广东独立储能披露 + 文献参数 | 是 | 用于奖励函数和约束 |

## 当前必须避免的表述

不要写：

- “本文使用了真实广东虚拟电厂 15 分钟运行数据。”
- “本文获取了完整国内电力现货出清数据。”
- “本文所有数据均来自真实公开数据。”

建议写：

- “本文基于国家能源局和广东市场公开披露数据，对虚拟电厂仿真场景进行参数校准。”
- “由于国内真实虚拟电厂运行数据难以公开获取，本文构建公开数据校准的仿真环境用于算法验证。”
- “文本事件参考电力市场公告、气象预警和需求响应通知的真实表达形式构造。”
