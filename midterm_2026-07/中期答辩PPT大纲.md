## 幻灯片 3：研究背景——新型电力系统下的 VPP 调度挑战

- "双碳"目标 + 新型电力系统 → 高比例新能源接入，源网荷储协同互动
- 虚拟电厂（VPP）聚合光伏、储能、可调负荷、电动汽车，参与市场与需求响应
- VPP 调度面临三重不确定性：负荷/光伏波动、市场价格波动、**外部文本事件**（气象预警、市场公告、需求响应邀约、新能源消纳提示）
- 关键痛点：文本事件先于数值效应出现，传统优化与仅数值状态 RL 难以利用

## 幻灯片 4：问题动机——文本信息盲区与 LLM 落地风险

- 现有 DRL 调度多仅消费数值状态 → 高温负荷压力、价格尖峰、消纳压力场景缺少前瞻性
- LLM 可理解中文事件文本，但**直接让 LLM 输出调度动作**存在风险：
  - 物理可行性（SOC/功率约束）
  - 可审计性与可复现性
  - 市场结算逻辑一致性
- 本研究立场：**LLM 作语义编码器，不作最终决策器**

*配图建议：左"数值状态RL盲区" vs 右"LLM直接控风险"对比*

## 幻灯片 5：研究目标与科学问题

- **总目标**：构建语言增强深度强化学习框架，提升 VPP 储能调度的事件感知能力
- **科学问题**：
  1. 如何将中文事件文本结构化为 RL 可用语义特征，且不丧失调度可复现性？
  2. 学习 Actor 与确定性语义先验如何分工？权重应否随事件覆盖变化？
  3. 当先验未覆盖的新事件类别出现时，LLM 语义特征能否帮助 SAC 适应？
  4. 上述适应性是否为仿真产物？能否在真实电价/气象下成立？

## 幻灯片 9：已完成工作三——LE-DRL-SAC 框架

- **DeepSeek 作语义编码器**：每事件模板查询一次并缓存，输出五维语义向量（风险/价格尖峰/负荷压力/消纳压力/储能偏置）
- LLM **不参与逐步控制**，无在线延迟，**不输出调度动作**
- 增强状态 $s^{aug}=[s^{num}, s^{sem}]$ 输入 SAC
- Actor 语义一致性正则：训练阶段促使 Actor 学语义对应行为
- 本地关键词编码器作 API 不可用时的可复现回退

## 幻灯片 10：已完成工作四——语义安全层与事件覆盖权重

- 测试期混合：$a^{final}=(1-w)a^{SAC}+w\,a^{sem}$，再经可行性裁剪
- **核心原则：权重 $w$ 视事件覆盖而定，非通用常数**
  - 已知事件：先验接近场景最优 → 高 $w$（主实验 $w=0.9$）
  - 未覆盖新事件：先验无对应分支 → 低 $w$，Actor 接管
- 数值安全层消融：纯数值先验 + 同公式 → 隔离"文本语义贡献"与"机制贡献"

## 幻灯片 11：阶段性结果一——跨场景基线对比

- 四场景（S1 常规/S2 高温/S3 价格尖峰/S4 消纳）×3 种子 ×80 episode
- 最终控制器 LE-DRL-SAC+安全层（$w=0.9$）：**−208,468.9 元**（最优）
- 优于 SAC-Numeric（−211,023）、Rule-Based（−208,970）、增强滚动时域（−209,988）、Linear-MPC（−210,656）、数值安全层 SAC（−209,125）
- 配对 bootstrap：vs SAC-Numeric +2,554 元，95% CI [+2,060, +3,044]，Wilcoxon p=0.0005
- 文本消融：SAC-Numeric ≈ LE-DRL w/o Text → 增益非来自维度增大

## 幻灯片 12：阶段性结果二——安全层权重扫描与机制隔离

- $w$ 扫描 0→1：收益单调改善，$w=0.9$ 为保留学习分量的混合配置
- 数值安全层：SAC-Numeric −211,023 → −209,125（机制本身有价值）
- 语义安全层：进一步到 −208,469（文本语义额外贡献）
- 结论：增益 = 安全层机制 + 结构化文本语义，二者可分离

## 幻灯片 13：阶段性结果三——新事件自适应（S5 负电价）

- S5：午间光伏过剩压低现货至负（最低 −210 元/MWh），晚高峰受抑
- 先验"晚高峰放电"分支失配；DeepSeek 正确识别过剩本质（消纳 0.90/偏置 +0.80）
- 松弛正则化（$\lambda=0$, $w=0$）+ S1–S5 重训练，5 种子：
  - LE-DRL-SAC：**−141,068 元**
  - SAC-Numeric：−146,550 元
  - **间隙 +5,482 元，95% CI [+4,875, +6,074]，5/5 种子有利**
- 行为：100% 负电价步充电，学会避免晚高峰放电

## 幻灯片 14：阶段性结果四——正则双重角色与参数变体稳健性

- 正则扫描：已知事件收益跨权重持平（<0.5%）；S5 随松弛单调改善 18,515 元

  - V2 趋正
- 结论：新事件自适应非单一参数设定产物，跨维度稳健

- **最接近真实市场的证据**

## 幻灯片 16：阶段性结果六——学习型事件覆盖门控

- 两个 SAC 专家（带/不带语义正则）+ 小门控网络 $g_\psi(s)\to w$
- 以**逐批归一化 Q 差异**为二分类目标训练，**无需覆盖标签**
- 结果：
  - 已知事件 S1–S4：平均 $w=0.65$（偏向正则化先验专家）
  - S5 + 变体 V1–V4：$w=0.31$–0.48（偏向自由专家）
- 结论：仅凭 Q 差异信号即可从数据恢复"已知高 $w$/新事件低 $w$"分割 → 覆盖相关权重可自动化

*配图建议：门控权重按场景分布图*

## 幻灯片 17：主要创新点

1. **解耦式语言增强 SAC**：LLM 严格作语义编码器（每模板查询缓存），不输出动作，决策确定可审计
2. **事件覆盖相关语义安全层**：权重 $w$ 随覆盖变化，数值安全层消融隔离语义/机制贡献
3. **新事件自适应**：先验未覆盖负电价事件，松弛正则下 LLM 特征助 SAC 适应（+5,482 元，5/5）
4. **真实电价+真实气象验证**：德国 DE-LU 真实价格 + Open-Meteo 真实气象（+2,201 EUR，4/5）
5. **学习型事件覆盖门控**：Q 差异无标签信号恢复覆盖分割，权重可自动化

## 幻灯片 18：存在问题与不足

1. 数据真实性边界：主轨迹为公开数据校准仿真，非实测遥测（OOD + 真实电价已部分缓解）
2. DeepSeek 相对关键词编码器的独特优势待专用对比坐实
3. 统计方法已修正：种子配对 bootstrap + Wilcoxon（旧版 3 均值 bootstrap CI 过窄已替换）
4. 完整 MILP/MPC 滚动优化基线在相同约束下对比仍需补
5. 门控仅演示覆盖分割可学，混合动作奖励最优化待做
